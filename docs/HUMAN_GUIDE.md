# Human Guide

## 概要
この拡張は ChatGPT のスレッド操作を補助します。削除、移動、定型文入力、サイドバー自動スクロールなどを設定画面からON/OFFできます。

## 使い方
1. `chrome://extensions` を開く
2. デベロッパーモードをON
3. 「パッケージ化されていない拡張機能を読み込む」からこのリポジトリを選択
4. 拡張の「オプション」を開いて必要な機能を有効化

## 主な設定
- 削除ボタン / 削除ショートカット
- 上下・ランダム移動（ボタン/ショートカット）
- 定型文5種（ラベル・本文・ショートカット）
- レイアウト位置（左下ボタン群位置、本文最大幅）
- 自動スクロール（間隔・回数・しきい値）

## トラブルシュート
- ボタンが出ない: 設定画面で対象機能がONか確認
- 動かない: ChatGPTタブを再読み込み
- 設定が反映されない: 保存後に数秒待ってから再読み込み
