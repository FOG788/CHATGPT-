# Human Guide

## 概要
この拡張は ChatGPT の会話操作を効率化するためのツールです。必要な機能だけを個別に ON/OFF できます。

## 対応ページ
- `https://chat.openai.com/*`
- `https://chatgpt.com/*`

## 導入手順
1. `chrome://extensions` を開く
2. デベロッパーモードを ON
3. 「パッケージ化されていない拡張機能を読み込む」からリポジトリ直下を選択
4. 拡張の「オプション」を開く
5. 機能を選んで「保存」

## 設定項目

### 削除
- 削除ボタン
- 削除ショートカット
- 削除後の移動先（ランダム / 上 / 下）

### 移動
- 上下移動ショートカット
- ランダム移動ショートカット
- ランダム移動ボタン
- 上下移動ボタン

### 定型文
- 5件まで設定可能
- 各項目は「ボタン名 / 本文 / ショートカット」で管理

### 表示
- ボタン群の表示位置（左・下）
- メイン文章の最大幅
- 定型文ボタンの横幅

### 自動スクロール
- 有効 / 無効
- 実行しきい値（最近件数）
- 実行間隔（分）
- スクロール後待機時間（秒）

## うまく動かないとき
- 会話ページ以外では UI が表示されません
- 設定変更後はタブ再読み込みが確実です
- ショートカットが効かない場合は、入力フォーカスや他拡張との競合を確認してください
