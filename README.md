# ChatGPT Thread Deleter

ChatGPT の会話画面に「削除・移動・定型文入力」などの操作を追加する、軽量な Chrome 拡張です。

## この拡張でできること

- 会話スレッドの削除（ボタン / ショートカット）
- 会話スレッドの移動（上・下・ランダム）
- 左サイドバーの「最近件数」表示
- 5つの定型文ボタン（ラベル・本文・ショートカット編集可）
- サイドバーの定期スクロール（一覧の読み込み補助）

## インストール

1. このリポジトリをダウンロード / クローン
2. Chrome で `chrome://extensions` を開く
3. 右上の「デベロッパーモード」を ON
4. 「パッケージ化されていない拡張機能を読み込む」を押す
5. **`manifest.json` があるディレクトリ**（このプロジェクト直下）を選択

## 初期セットアップ

1. 拡張の「詳細」→「拡張機能のオプション」を開く
2. 使いたい機能だけ ON にする（初期値はすべて OFF）
3. 「保存」を押す
4. ChatGPT タブを再読み込み

## 主要機能

### 1) 削除
- 現在開いている会話を非表示化（ChatGPT 側 API の `is_visible=false`）
- 削除後の遷移先を `ランダム / 上 / 下` から選択

### 2) 移動
- 「上」「下」「ランダム」移動ボタン
- キーボードショートカットにも対応

### 3) 定型文
- 5つのスロットを自由に編集可能
- ボタン押下で入力欄へ挿入
- スロットごとにショートカット設定可

### 4) 自動スクロール
- 最近件数がしきい値以下のとき、サイドバーを一定間隔で自動スクロール
- 無駄な連続再実行を抑えるため、次回実行時刻を保持

## トラブルシュート

- UI が表示されない
  - 会話ページ（`/c/...`）を開いているか確認
  - 設定画面で対象機能が ON か確認
- 設定変更が反映されない
  - 「保存」後にタブを再読み込み
- 読み込みエラー
  - `manifest.json` の**ある階層そのもの**を読み込む
